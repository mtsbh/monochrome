Add to Playlist — 3 Approaches

Pick the direction that feels right. You can mix ideas from multiple options.

A

Enhanced List + Search Bar

Same compact popup, but adds a search/filter input at the top. Type letters to instantly filter matching playlists. Sort toggle (A–Z, Date Created, Recently Used). Each row shows cover thumbnail + name + track count. Clean, fast, keyboard-friendly.

Best for: knowing what you want, fast keyboard workflow

B

Grid View with Cover Art + Search

Popup opens slightly wider and taller. Playlists shown as a 2-column card grid with cover art, name, and track count. Search bar at top filters in real-time. Sort options. Good for browsing visually while still having search when you know what you want.

Best for: browsing + search, visual recognition

C

Tabbed: Recents + All + Search

Compact popup with 3 tabs: "Recent" (last 5–8 playlists you added to), "All" (full sorted list), and a search icon that expands a search bar. Most used playlists are one tap away. Sort on the "All" tab.

Best for: repeat workflows, quick access to favorites

My recommendation

I'd suggest B (Grid + Search) since you have 30–40+ playlists and browse visually. The grid makes covers recognizable at a glance, and search handles the "I know what I want" case. We can keep it compact by limiting the grid to a scrollable area. Option A is the safer/simpler fallback if you want minimal visual change.
